Counter & Timer

Приложение счётчика и таймера. Создано на основе фреймворка React. Значение счётчика можно изменять, как при помощи кнопок приложения, так и при помощи клавиш "стрелка вверх" и "стрелка вниз" на клавиатуре.

Таймер осуществляет отсчёт количества секунд, указанных в соответствующем поле ввода. Интервал отсчёта в миллисекундах можно задать в соответсвующем поле. По окончанию отсчёта появляется окно с сообщением о завершённой работе. Кнопка сброса обнуляет все значения таймера.

Применённые технологии:

  • HTML
  • CSS
  • JavaScript
  • React
